# CN_Project
Networked Multiplayer Game

- Sockets in JAVA is used for networking connections along with threads for handling multiple clients simultaneously.
- Game -> Hangman
- 2 players. One provides the word (chooser) other guesses it (guesser) ,by giving a letter in each try, within 6 tries.
- if all letters in the word are found,the guesser wins, else the chooser wins.
